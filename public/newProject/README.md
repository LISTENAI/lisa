# 说明

### 固件开发打包烧录流程
```
1、config/vui里存放固件开发配置，填上必要的配置参数。
2、点击project.csk右上角进行打包。
3、打包成功后，左边栏点击历史版本，选择固件版本，点击进行烧录。
```

# 项目结构

```
.
├── config                            // 存放项目配置文件
│   └── vui														// 存放可视化配置文件
│       ├── base.csk							  	// 基础配置
│       ├── tones.csk									// 提示音
│   		└── interact.csk              // 交互配置(欢迎、唤醒、命令词)
├── deps                              // 存放项目依赖文件
│   └── tones_include									// 存放导入提示音
├── doc                               // 存放项目文档文件
│   └── 硬件准入规范.xlsx								// 硬件准入规范
├── target                            // 存放项目产物（新建项目无该输出产物目录）
│   ├── .building								      // 固件打包中间产物
│   └── build                         // 存放固件版本
├── test                              // 存放项目测试文件
├── project.csk                       // 项目信息
.

```
